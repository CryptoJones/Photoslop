# Keyboard Shortcuts

## Tools
`V` Move 路 `M` Marquee 路 `L` Lasso 路 `Shift+L` Poly Lasso 路 `Alt+L` Magnetic 路
`W` Wand 路 `Shift+W` Quick Select 路 `B` Brush 路 `Shift+B` Pencil 路 `E` Eraser 路
`G` Bucket鈬凣radient (cycles) 路 `S` Clone 路 `Shift+S` Healing Brush 路
`J` Spot Heal 路 `Shift+J` Smudge 路 `Alt+Shift+J` Patch 路 `O` Dodge 路
`Shift+O` Burn 路 `Y` Liquify 路 `Shift+Y` Puppet 路 `Shift+P` Perspective 路
`U` Shape (`Shift+U` cycles rect/ellipse/line) 路 `P` Pen 路 `T` Text 路
`I` Eyedropper 路 `C` Crop 路 `H` Hand (`Space` momentary) 路 `Z` Zoom 路
`R`/`Shift+R` Rotate View

## Brush
`[` / `]` size 路 `Shift+[` / `Shift+]` hardness 路 `X` swap fg/bg 路 `D` reset colours

## Edit
`Ctrl+Z` / `Ctrl+Shift+Z` undo/redo 路 `Ctrl+A` select all 路 `Ctrl+D` deselect 路
`Alt+Backspace` Fill Layer 路 `Shift+F5` Content-Aware Fill 路
`Ctrl+Alt+D` Feather Selection 路 `Ctrl+Alt+R` Refine Selection 路 `F9` Play Action

## Layers

`Ctrl+Alt+G` clipping mask 路 `Ctrl+E` merge down 路 `Ctrl+Shift+E` merge visible 路
`Ctrl+Shift+Alt+E` stamp visible

## Adjustments
`Ctrl+L` Levels 路 `Ctrl+M` Curves 路 `Ctrl+U` Hue/Saturation 路 `Ctrl+B` Color Balance

## Image & transforms
